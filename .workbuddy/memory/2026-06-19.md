## Button 事件转发修复 (8597b38)
- **根因**：Button.vue Props 声明了 onClick/disabled，Vue 3 当作组件 prop 消费，不再透传给 reka-ui Primitive → 原生 button 收不到事件
- **修复**：从 Props 移除 onClick/disabled，Home.tsx 用 `// @ts-ignore` 绕过 TSX 类型限制
- 修正 drag-drop 事件数据解析路径，添加调试日志
- 绑定文件 wails3 dev 重新生成为 .ts
