# Electron 图片处理项目功能列表

## 项目概述

这是一个基于 Electron + Vue 3 + TypeScript 开发的多媒体处理工具，主要用于图片和视频的批量处理、格式转换和压缩。

## 核心功能模块

### 1. 图片处理功能

#### 1.1 x8u DNG 转换

- 将 DNG 格式图片转换为 JPG/PNG/AVIF/WebP
- 支持批量处理和队列管理
- 可配置输出质量（默认100）
- 自动导出到原目录的 `x8u_convert_jpg` 文件夹

#### 1.2 图片格式转换

- 支持转换为 WebP、AVIF、HEIF 格式
- 可调节压缩质量（默认80）
- 支持导出为压缩包选项
- 批量处理多个图片文件

#### 1.3 递归图片转换

- 递归转换目录下所有图片
- 保持原有目录结构
- 输出到输入目录同级的新目录
- 可选择保持原文件后缀或使用目标格式后缀
- 支持跳过已是目标格式的文件
- 实时显示转换进度和统计信息
- 可配置压缩质量（默认90）

### 2. 视频处理功能

#### 2.1 视频压缩

- 支持多种视频格式的压缩
- 硬件加速支持（NVIDIA、AMD、APPLE）
- 可配置码率（默认20000）和分辨率系数（默认1）
- 支持同步修改时间（压缩后文件时间与原文件一致）
- 实时显示压缩进度和剩余时间

#### 2.2 视频转换为 H.265 MP4

- 将任何视频格式转换为 H.265 编码的 MP4
- 支持保持原视频码率（码率设为-1）
- GPU 硬件加速支持
- 可配置分辨率系数

#### 2.3 TS 文件转 MP4

- 将 TS 文件转换为 MP4 格式
- 自动过滤非 TS 格式文件
- 保持原视频质量不变
- 转换后文件保存在原目录的 `ts_to_mp4` 目录

#### 2.4 INSV 视频压缩

- 专门用于 Insta360 INSV 格式视频压缩
- 支持 GPU 硬件加速
- 可配置码率（默认42000）和分辨率系数

#### 2.5 OSV 视频压缩

- 用于压缩大疆全景相机视频
- 提供两种压缩方式：
  - 标准压缩
  - 基于文件格式的智能压缩（适合非标准格式）
- GPU 硬件加速支持
- 可配置码率和分辨率系数

#### 2.6 视频文件信息查看器

- 查看视频详细信息
- 显示所有数据流信息
- 包含格式、时长、大小、比特率等详细参数
- 显示每个流的编解码器、分辨率、配置文件等信息

#### 2.7 合并音视频

- 合并拥有公共前缀的两个视频
- 自动将旧视频移动到 `delete_mergeAudioVideo` 目录

#### 2.8 全景视频帧提取与拼接

- 支持 INSV/OSV 全景视频文件
- 根据时间码提取指定时间点的帧
- 自动提取前后画面并左右拼接
- 支持输出为 PNG（高质量无损）或 WebP（尺寸更小）格式
- 时间码格式：00:00:00.000（时:分:秒.毫秒）

### 3. 文件管理功能

#### 3.1 批量重命名

- 递归重命名文件夹所有文件
- 支持按创建时间或修改时间重命名
- 批量处理大量文件

#### 3.2 Mac 缓存文件清理

- 移动 Mac 生成的 ".\_" 缓存文件
- 自动移动到根路径下的 `mac_catch` 目录
- 递归处理所有子目录

## 技术特性

### 硬件加速

- 自动检测 GPU 类型（NVIDIA、AMD、APPLE）
- 支持硬件加速视频编解码
- 提升处理速度和效率

### 队列管理

- 所有批量操作支持队列管理
- 实时显示处理进度
- 显示剩余时间估算
- 支持多任务并行处理

### 用户界面

- 拖拽上传文件
- 实时配置参数
- 进度可视化
- 友好的错误提示

## 支持的文件格式

### 图片格式

- 输入：DNG、JPG、PNG、AVIF、WebP、HEIF 等常见图片格式
- 输出：JPG、PNG、AVIF、WebP、HEIF

### 视频格式

- 输入：MP4、MOV、AVI、INSV、OSV、TS 等常见视频格式
- 输出：MP4（H.264/H.265）

## 配置选项

### 图片处理配置

- 压缩质量：0-100
- 输出格式选择
- 是否导出压缩包
- 是否保持原文件后缀

### 视频处理配置

- 码率设置（kbps）
- 分辨率系数
- GPU 硬件加速选择
- 是否同步修改时间

### 文件管理配置

- 重命名方式（创建时间/修改时间）
- 帧提取时间码
- 帧输出格式

## 输出目录规则

- x8u DNG 转换：`原目录/x8u_convert_jpg/`
- TS 转 MP4：`原目录/ts_to_mp4/`
- 合并音视频删除文件：`原目录/delete_mergeAudioVideo/`
- Mac 缓存文件：`根路径/mac_catch/`
- 递归图片转换：`输入目录同级/新目录/`
- 其他处理：通常在原目录或原目录的子文件夹

## 项目优势

1. **批量处理能力强**：支持大量文件的批量处理
2. **硬件加速**：充分利用 GPU 加速，提升处理速度
3. **格式支持全面**：支持主流图片和视频格式
4. **操作简单**：拖拽上传，参数可视化配置
5. **进度可视化**：实时显示处理进度和剩余时间
6. **专业功能**：支持全景视频、DNG 等专业格式处理
